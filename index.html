Firestore Setup Required

Open index.html in a code editor and replace the placeholder values in the firebaseConfig object with your Firebase project credentials. 

const firebaseConfig = {
  apiKey:            "YOUR_API_KEY",
  authDomain:        "YOUR_PROJECT.firebaseapp.com",
  projectId:         "YOUR_PROJECT_ID",
  storageBucket:     "YOUR_PROJECT.appspot.com",
  messagingSenderId: "YOUR_SENDER_ID",
  appId:             "YOUR_APP_ID"
};

1. Go to https://console.firebase.google.com
2. Create a project → Enable Firestore only (No Auth needed!)
3. Project Settings → Your Apps → Web → Copy config
4. Paste above and save the file

No Firebase Authentication required — login is handled locally, no billing needed.
